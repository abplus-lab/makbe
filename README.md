# makbe

makbeとは、「Modulo Architecture の KeyBoard は Eよね！」の短縮形で、壺を愛でる大佐とは関係ありません。

冗談はさておき、**makbe** はModuloアーキテクチャに基づいて設計されたキーボードのファームウェアフレームワークです。

### 最初のGOAL

examplesにあるように、Seeeduino XIAO で4x4のキーパッドを使えるようにします。
